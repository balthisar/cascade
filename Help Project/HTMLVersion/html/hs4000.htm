 This section briefly highlights the interface to the Cascade application, and its overall operation. It is not meant to serve as a tutorial; please see the printed documentation for that.
The following picture shows the main interface to the application, with red labels to indicate the major portions of the window. The paragraph below the illustrations has the corresponding window portions highlighted in matching red.

  • CSS2 classes are created using the main menu.
  • The selectors for the classes you create are placed into the class list.
  • The attributes for the class selected in the class list are manipulated in the 14 attributes panes.
  • A preview of the CSS2 code for the class selected in the class list is shown in the preview pane, and reflects the attributes' values from the 14 attributes panes.
  • You can work with multiple documents. To switch between open documents, select the document in the documents tabs.
  • Often-used main menu functions are duplicated for quick access in the shortcuts bar.
