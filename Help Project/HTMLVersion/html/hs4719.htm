Preview in Browser...: This menu command allows you to see what your stylesheet looks like when applied to the HTML file of your choice. Your HTML file must use the classes you create. For example, if your Cascade document has an Element Class <P>, then chances are your HTML file already uses it. But if your document has a class called P.green, it will not show up in your HTML file, unless the HTML tries to use <P.green> somewhere. Cascade create temporary files for this work, so none of your original files will be touched. Cascade tries to insert the proper <LINK> tag into the HTML file, so that the HTML will load the stylesheet in the browser. Please read this warning.
The preview will occur in whichever application is registered on your system as handling HTML files (i.e., your default web browser).
