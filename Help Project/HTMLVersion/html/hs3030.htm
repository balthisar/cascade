Subclasses

 Like a class of type "Class," subclasses in CSS2 and Cascade are applied to HTML elements by the user agent. Whereas a class can be applied to any element, subclasses are designed for one element specifically. The selector for a subclass takes the form "ELEMENT.subclass" (where ELEMENT is the HTML element you would subclass). For example, if...

the stylesheet has:
P.subclass {font-family: "Helvetica"}
 and the HTML has:
<P.subclass>

 ...then the <P.subclass> becomes a new element, in a manner of speaking. This HTML tag will then display the content with all of the normal CSS attributes for the <P> element, except for those which have been overridden in the subclass; in the above example, the <P.subclass> tag's content would be rendered using the font "Helvetica" instead of <P>'s default font.

To create new subclass in Cascade, make sure that an Element class is selected in the class list, and execute the Classes/Subclass... command from the main menu, or select Subclass... in the class list's contextual menu.
