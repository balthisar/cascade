List-Style-Image

Table of Contents Balthisar Software Site Attributes Panes


The list-style-image attribute sets the image that will be used as the list item marker. When the image is available, it will replace the marker set with the 'list-style-type' marker.

For example, the following sets the marker at the beginning of each list item to be the image "ellipse.png".
UL { list-style-image: url("http://png.com/ellipse.png") }
