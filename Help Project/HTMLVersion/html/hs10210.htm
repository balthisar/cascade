Text-Indent

Table of Contents Balthisar Software Site Attributes Panes


The text-indent property specifies the indentation of the first line of text in a block. More precisely, it specifies the indentation of the first box that flows into the block's first line box. The box is indented with respect to the left (or right, for right-to-left layout) edge of the line box. User agents should render this indentation as blank space.
The value of text-indent may be negative, but there may be implementation-specific limits.
