Visibility

Table of Contents Balthisar Software Site Attributes Panes


The visibility attribute specifies whether the boxes generated by an element are rendered. Invisible boxes still affect layout (set the display property to 'none' to suppress box generation altogether). Values have the following meanings:
visible
The generated box is visible.
hidden
The generated box is invisible (fully transparent), but still affects layout.
collapse
If used on elements other than rows or columns, 'collapse' has the same meaning as 'hidden'.
