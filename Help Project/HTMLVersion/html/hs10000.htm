 This next section explains all of the attributes which are programmable in CSS2 and in Cascade. It is not intended to be a replacement of the CSS2 Specification from W3C, but rather to serve as a quick reference to the CSS attributes.

 Note that in most of the CSS attributes, the value can be set to Inherit. For simplicity and space, this value will not be mentioned in any of the attributes' descriptions. Setting an attribute's value to inherit means that the class will inherit the settings from its ancestor class, if applicable.
