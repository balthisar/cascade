What's in Store

 The future of Balthisar Cascade is looking like this right now:
  • Version 2.0b9 will need to have its bugs fixed, if there are any remaining, and if the beta-testers report them faithfully.
  • A live-preview plug-in is planned. This will enable Cascade users who have IE4+ installed on their systems to see near-instant results of their stylesheet modifications (the Cascade application currently allows a preview in browser feature which works in conjunction with existing HTML markup).
  • An import-css plug-in is in development. This will allow Cascade to import existing CSS files, and to parse existing HTML files into Cascade.
  • Eventually, version 2.0 will move out of beta, which means I will no longer distribute the application for free. If you still have the beta version at that time, then it's yours to keep and use forever (this program will not expire). But if you enjoy the application and want continued support, you may want to consider obtaining the shareware version in the future. There is no anticipated date for this transition yet -- the application must be "perfected" first.
  • The shareware version will be released. This application and its help file are the result of over 1200 hours of work. At an average programmer's pay, you can do the math and figure out its value. Because the shareware version will likely cost US$10-$20, I would have to receive 2,250 to 4,500 paid registrations to "break even." As you know, that is probably a ridiculous number, and you should see that I really do all this work for the mere love of it.
  • New! The Freeware/Shareware plan has been finalized! A single version of Cascade will be released that is both Shareware and Freeware. Users can choose to register if they care to, and they won't be made to feel guilty if they don't (i.e., there will be no annoying, constant reminders except for the first time the program is run). The Freeware/Unregistered version will lack only one feature -- the ability to use plug-ins.
