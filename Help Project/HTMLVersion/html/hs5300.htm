Cutting, Copying, and Pasting

Table of Contents Balthisar Software Site Cascade Tips...


 In the class list, you can cut, copy, and paste classes.

Whenever you cut or copy a class in the class list, the class is copied onto the Windows clipboard, where Cascade and other applications can use the data that is present there. Whenever you cut or copy, any information already present on the clipboard is replaced.
Furthermore, Cascade places the data onto the clipboard in two formats: text format and Cascade format. This means, after copying or cutting a class in the class list, you can switch to another application, and paste the class's CSS text. You can also paste the class into the class list.

When you cut a class, in addition to the items mentioned above, the class is removed from the class list, and hence, your document.

When you paste a class in Cascade, the application checks to see if the selector already exists. If not, the class is pasted immediately. If the selector already exists in the current document's class list, you are asked to supply a new selector for the class; the class will then be pasted as a Custom/Expert class.

When you paste over a class in Cascade, you apply the settings of the class in the clipboard to the currently selected class in the class window, replacing any attributes' values you may have set.
