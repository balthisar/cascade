Paste: This menu option pastes a new class and its attributes into the document. It is only available if you have previously cut or copied a class from the class list. The pasted data still remains on the clipboard, allowing you to paste multiple times. If the selector of the class on the clipboard does not already exist in the current document, then the new class will have the selector; otherwise, you will be asked to provide a new selector for the class, and the class will be a Custom/Expert class.
