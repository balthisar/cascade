Drag-and-Drop

Table of Contents Balthisar Software Site Cascade Tips...


 Cascade lets you use drag-and-drop to accomplish two different things:
  • Set cascading order in the class list.
  • Copy classes from one document to another.

    In the     class list, cascading order is determined by the order the selectors appear from top to bottom. To rearrange the class list, simply grab the selector you would move, drag it to the area of the list you want it to appear, and drop it there. The list will be rearranged to submit to your will.

    To make a duplicate of a class in the     class list, you can use drag-and-drop. In the source document, simply grab the class you would duplicate, and drag it to the document tab of the destination document. The source document and the destination document can be the same document, which results in duplicating the class in the same document.
    If the     selector does not exist in the destination document, the dropped class will be inserted automatically. If the selector does exist, Cascade will ask you for a new selector, and the class will be inserted as a Custom/Expert class.
