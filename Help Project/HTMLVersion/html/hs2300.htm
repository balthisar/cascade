 "Beta" software is not really "free" software. By using the software, you have agreed to a license. The license does not require that you, the user, report information to the application developer.
However, beta software is released free-of-charge for the purpose of soliciting feedback about the quality of an application. Some large mega-corporations release program suites, and even entire operating systems, for the purpose of beta-testing, without informing the user, and charging anywhere from $100 for a simple Windows-based operating system to $500 for a full Office suite. These same companies then have the nerve to charge the user additional money for "telephone support" in order to report the bugs!
Part of a beta-tester's moral obligation is to report bugs and suggest ideas for the application. This is a meager pittance for free use of a valuable application. So, dear user, please do your part!
Submitting feeback is especially easy if you have a connection to the internet, which you most likely do, since you are using this application. Please see Contacting Balthisar Software for simple contact information.
If you do find that Cascade experiences problems, please remember to tell us how to duplicate the problem, step-by-step. If the problem only occurs in certain files, go ahead and attach the file in an email letter to us. This help file can probably also benefit from criticism, as well. If you notice a grammatical error, an invalid or malfunctioning link, etc., please feel free to report it!
