 This section merely shows you what the documents tabs are, as well as to tell you about a special feature related to drag-and-drop and the documents tabs.



 These tabs are used to switch between open documents, if you have more than one document open. Because each document is cached to disk in multiple-document situations, documents with a larger number of elements will take longer to switch back-and-forth between. This is normal behavior, and no cause for alarm.

An added feature of the documents tabs is the ability to copy classes from one document to another through drag-and-drop. All you have to do is start dragging the class you wish to duplicate from the class list. Then, drop the class on top of the document tab you wish to copy the class to. You can even copy classes within the same document by dropping the class on its own document's tab!
If the selector of the dropped class does not already exist in the destination document, then the new class will be an exact copy of the original. If the selector already exists, Cascade may ask you to type the new selector, which will make the new class a Custom/Expert class(see Preferences to determine whether you will be asked to rename the class).
