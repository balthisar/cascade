Creating a Default Set of Elements

Table of Contents Balthisar Software Site Practical Use


 If you constantly create stylesheets with many of the same classes, it may be a good idea for you to create a startup set, using stationary. To do so, just create your default stylesheet as you would any stylesheet, but when you save it, be sure to selected "Cascade Stationary" as the file type, and name the file "default" -- resulting in a file called "default.WWTD" on your hard drive. For this to work as a default set of classes, the file must reside in the same directory as the Cascade application.
