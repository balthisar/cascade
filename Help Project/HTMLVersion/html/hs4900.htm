 This section explains how to use the HTML Color Picker, which is invoked any time you press the ellipsis button of a Combo Editor designated for colors.
The HTML Color Picker used by Cascade gives you many choices, and has the following features:
  • The ability to use CSS2's new color names to specify system colors, which will render in the appropriate color on different users' operating systems.
  • The standard ability to mix RGB colors and specify them in hexidecimal format.
  • The ability to select HTML 3.2+ color names from hue-oriented lists.
  • A custom colors palette that remembers your custom colors forever.
  • Auto-pane-switching in custom color implementation.

    The color picker is divided into two primary panes, which operate completely differently, depending upon whether or not you want an     HTML color or a System color.
