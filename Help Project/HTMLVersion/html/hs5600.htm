Variations on Creating New Documents

 When you use the File/New... command in the main menu, or use the new file shortcut, the new document will be created based on "default.WWTD" if it is present in the same directory as the Cascade executable.
If you use the Ctrl-N keypress, then Cascade will create the new document with no classes at all (i.e., it will ignore the presense of "default.WWTD").
