Tag/Untag: This menu option changes allows you to change the current "tagged" state of the selected class in the class list. When a class is tagged, the class list shows the class name in bold; otherwise, the class name is not bold.
By tagging some classes, and not others, you are provided some flexibility in your preview and export functions during the development of your stylesheets.
Note: @media descriptors cannot be untagged.
An alternative to using this button to change the tagged state of a class is to use the Tag/Untag Button, or the main menu
