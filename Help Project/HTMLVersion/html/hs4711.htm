New...: This menu command will create a new, untitled document. If the file called "default.wwtd" (a stationary, or template, document) exists in the same directory as the Cascade application, then the new document will be based on that stationary.
Note: if "default.wwtd" does exist, and you want a blank document instead of a new one based on the stationary, then don't use the New menu command. Instead, type Ctrl-N to create a new document, which forces Cascade to ignore the default stationary.
