 The HTML Color Picker is divided into two separate, primary panes because depending on the type of color you wish to choose, there are different requirements. The section explains the standard picker for HTML colors, illustrated below.
The color picker is composed of two main parts: the composite, or "Current Color" portion on the right, and the color picker portion on the left, which is divided into three panes which allow you to pick colors differently.

The composite, or "current color" portion is used to show you the current color you have chosen, and is the color that will serve as the value of the attributes whose color you are modifying. Above the color is the RGB value of the color depicted. Below the color is the HTML color name, if there is one, of the chosen color.
