Shortcut Bar
Table of Contents Balthisar Software Site Cascade Overview


The shortcut bar provides one-click access to some of the more common functions found in the main menu.
The shortcut bar can be dragged from its position at the top of the main window, and placed anywhere on any of your monitors. It can also be snapped back by dragging it back over the main window.
If you hover the mouse over the shortcut bar icons, a hint window will tell you the purpose of the shortcut. Additionally, the following items show you the meaning of the shortcut bar icons.

 Creates a new Cascade document.

 Opens an existing Cascade document.

 Saves the current Cascade document.

 Cuts the currently selected class from the class list, and puts it on the Windows clipboard.

 Copies the currently selected class in the class list onto the Windows clipboard.

 Pastes the current clipboard selection to the document as a new class.

 Deletes the currently selected class from the class list without putting it onto the Windows clipboard.

 Previews the current document in the default web browser with the HTML document of your choice.

 Drops a short menu with options to allow the export of the current Cascade document. Choices are all classes, tagged classes, and selected class.

 Applies the name of any CSS file to any HTML documents of your choice.

 Puts the system cursor into context-sensitive-help-mode, which allows you to click on an item to get help about it. To get context help for menu items, hover the cursor over the menu item, and press Ctrl-F1.
