Pseudo-classes of Subclasses

Table of Contents Balthisar Software Site Cascade Classes


 Cascade distinguishes between a run-of-the-mill Pseudo-class and a Pseudo-class of a subclass in order to better support novice users. Just as any Element class can have associated Pseudo-classes, subclasses can have Pseudo-classes as well. The selector takes the form "ELEMENT.subclass:pseudo" (where ELEMENT is a valid HTML Element). For example:

A.framelinks:visited {Color: red}

 If the above style rule is included with your HTML document, any time the <A.framelinks> tag is used, visited links will be rendered in red. However, because in this example we didn't create a pseudo-class for the <A> Element class, your <A> tags' content will be rendered according to the user agent's default stylesheet.

To create new pseudo-class or pseudo-element of a subclass in Cascade, make sure that a subclass is selected in the class list, and execute the Classes/Pseudoclass... command from the main menu, or select Pseudoclass... in the class list's contextual menu.

Please the the Pseudo-Classes and Pseudo-Elements topic for more information.
