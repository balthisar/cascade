 An "Element Class" in Cascade refers to a class that applies to an HTML element at all times. In way of example, consider how the <P> (paragraph) element is typically rendered in your web browser. In most browsers, <P> text looks very similar to the text you are now reading. This is because your web browser's default style sheet has a element class called P which defines the way <P> text looks. Suppose we created a stylesheet which looks like this:

P { font-family: "Arial" }

 Now, any time we use the <P> element, our paragraphs would look more like this one, because we created an Element Class, that is, a class that applies to an HTML element.

 To create new element classes in Cascade, use the Classes/New Element... command in the main menu.

Note: in Cascade, all Element Classes are forced into all capitalization. This helps distinguish them visibly from other types of classes. In addition, the user is permitted to type almost any HTML element name, even elements which do not exist in HTML. This keeps Cascade from becoming outdated, but it allows typos.
