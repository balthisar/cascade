Apply Stylesheet to HTML File(s): This menu option will attempt to make any HTML files you specify compatable with the stylesheets you design and export from Cascade. You will first be asked to select all of the HTML files you would like to be made compatable. You can select as many files in the same folder as you like at one time, but bear in mind that you will apply the same stylesheet to all of them. Run this tool multiple times if you need to apply different stylesheets to different files. After selecting files, you will be asked to specify the name of the stylesheet you would like to apply. This is the name of the CSS file as it will exist on the server or locally on your system, including any necessary path information. When you have specified the CSS file, Cascade will then modify your HTML files by inserting a <LINK> tag into the <HEAD> section of each HTML file. Please read this warning.
