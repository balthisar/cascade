Voice-Family

Table of Contents Balthisar Software Site Attributes Panes


The voice-family attribute specifies a comma-separated, prioritized list of voice family names (compare with 'font-family'). Values have the following meanings:
<generic-voice>
Values are voice families. Possible values are 'male', 'female', and 'child'.
<specific-voice>
Values are specific instances (e.g., comedian, trinoids, carlos, lani).
