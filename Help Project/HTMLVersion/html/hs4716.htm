Export Tagged Classes...: This menu command allows you to export your Cascade documents as CSS code or HTML code. Alone, a Cascade document is useless -- it must be exported for use on the internet. Choosing this command will present to you a file save-as dialogue, where you can choose the file format as being HTML Format or CSS Format, as well as provide a name for the file. Cascade will next translate the classes in the class list which are tagged into the appropriate format, and create the file you asked for.
