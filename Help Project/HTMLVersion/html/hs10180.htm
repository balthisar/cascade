The CSS2 specification supplies the font attribute as a shortcut for setting some of the other font-related properties. This aspect of the font attribute is not supported by Cascade, since Cascade performs the exact same functionality in using the longhand form, and the shortcut syntax would not save the user any amount of time or keystrokes. However, CSS2 also specifies an alternate meaning of the font attribute, which allows the use of system fonts. Values for the font attribute therefore can be as follows:

caption
 The font used for captioned controls (e.g., buttons, drop-downs, etc.).

icon
 The font used to label icons.

menu
 The font used in menus (e.g., dropdown menus and menu lists).

message-box
 The font used in dialog boxes.

small-caption
 The font used for labeling small controls.

status-bar
 The font used in window status bars.
