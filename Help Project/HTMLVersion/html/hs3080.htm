Custom/Expert Classes

Table of Contents Balthisar Software Site Cascade Classes


 Custom/Expert classes are a Cascade contrivance designed to provide the stylesheet author the ability to create a class using any valid CSS2 selector. The "ready made" class types provided by Cascade may not meet all of the requirements an advanced stylesheet may have. For example, CSS2 provides for a mixture of two types of contextual classes, as shown here:

DIV OL > LI P {color: red}

 The example can only be created as an Expert/Custom class. It combines two types of contextual selectors: descendent selectors (the built-in Cascade contextual selector type), and child selectors. The example gets quite precise as to when the <P> element will be rendered in red. In this case, <P> will use the specified attribute (color) only when it is nested somewhere in a <LI> (list item) tagset, and if that <LI> tag is nested immediately within an <OL> (ordered list) tagset, and if that <OL> tagset is nested somewhere in a <DIV> tagset. So...

<DIV>
<OL>
<TABLE>
<TR>
<TD>
<LI>
<P>Hello, world!
...</DIV>

 ...does not satisfy the example selector, so "Hello, world!" would be rendered in its default color. The spoiler causing the color attribute not to have an effect is that the <LI> is not an immediate child of an <OL>. However, this HTML...

<DIV>
<OL>
<LI>
<TABLE>
<TR>
<TD>
<P>Hello, world!
...</DIV>

 ...would render "Hello, world!" in red, because the <LI> is an immediate child of the <OL>. The <P> doesn't have to be an immediate child of the <LI>; it's enough that it's merely a descendent.

Another use of expert selectors is specification of multiple selectors which allow the attributes you set to apply to multiple classes. For example:

H1, H2, H3, H4, H5, H6 {color: red; font-family: Helvetica}
H6 {font-family: Times}

 The above setup allows rapid specification of color and font for all <Hx> elements. As an added demonstration, the second line above changes the font family for just <H6>, while still keeping the color red from the first line.

As you can see, using Custom/Expert Selectors gives you tremendous power over your stylesheets. To create a Custom/Expert selector, execute the Classes/Expert Selector... command from the main menu. Also see the W3C's CSS2 Specification Document for all the information you could ever possibly need about the creation of selectors.
