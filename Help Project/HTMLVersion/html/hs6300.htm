Saving a Document

Table of Contents Balthisar Software Site Practical Use


 When you have created your stylesheet, it's usually a good idea to save it to the hard drive in case you need to make changes later. To save a file, use the main menu to select either the File/Save or the File/Save As... command.
If you choose File/Save As..., or File/Save, and the file hasn't been saved yet, you will be presented with the following dialogue box.



 At this point, you can choose to save the file as a Cascade document, as a Stationary document, as a CSS file, or as an HTML file. The latter two options export your document for use with your HTML files.
