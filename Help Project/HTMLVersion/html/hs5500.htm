 Cascade currently has limited support for stationary documents, also known as templates. Cascade stationary files are identical to Cascade documents, with the exception of the file extension. Stationary files use the extension .WWTD instead of .WWS2.
When you save your file, you can use the save-as dialogue to change the file extension so that the document will saved as a stationary file.
Whenever stationary files are opened in Cascade, you will be able to alter the stationary file.
Whenever stationary files are opened from the shell (i.e., they are double-clicked to start), they will open as new, untitled documents (e.g., "Document 1"; "Document 2").
