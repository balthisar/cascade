Borders Attributes Pane

 The properties in the Borders Attributes pane specify the width, color, and style of the border area of a box. These properties apply to all elements. Note: Notably for HTML, user agents may render borders for certain elements (e.g., buttons, menus, etc.) differently than for "ordinary" elements.
