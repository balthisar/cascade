Name Picker

Table of Contents Balthisar Software Site


 The name color picker lets you select a human-readable color by name, as defined in the W3C's HTML specification.
To ease your choice of color selections, the name picker has the available colors separated into groups by hue. To select a named color, first use the radio buttons to select a color group, then use the drop-down list select the color you desire. After doing so, the composite color will update to show you what the color looks like.

If you manage to drag the composite color over the name picker, the name picker will temporarily disappear to show the custom color picker, so that you can drop the color there. Once you are done with the drag-and-drop, the name picker will reappear.
