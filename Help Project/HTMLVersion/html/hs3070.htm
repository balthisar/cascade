 ID classes in CSS2 and Cascade are implemented similarly to classes. Just as elements may specify a CLASS attribute within the tag to utilize a class, elements can also specify an ID attribute within the tag to utilize an ID class. ID classes take the form "#classname" (note the pound/hash sign preceding the name). For example, if...

the stylesheet has:
#xyz {color: red}

 and the HTML has:
<P ID=xyz>Hello, world!

 ... then the phrase "Hello, world!" would be rendered using all of the default styles of <P>, except for what the class overrides, in this case, the color. "Hello, world!" would be rendered in red. Notice that in HTML, the ID attribute value for elements does not include the initial pound/hash sign.

To create new ID class in Cascade, execute the Classes/New ID... command from the main menu.
