The font-size property describes the size of the font when set solid. Cascade allows the follow types of size specifications, as shown in the image to the left. These are:

Absolute Sizes, selected with a pop-up menu, are known to the user agent. Possible values are:
xx-small x-small small
medium large x-large
xx-large

Relative Sizes, selected with a pop-up menu, are interpreted by the user agent and scaled in relation to the size of the parent element. Possible values are:
larger smaller

Length Sizes and Percentage Sizes are both selected using the Spin Editor and its associated po-up menu.

Length Sizes specify an absolute font size.

Perecentage Sizes specify an absolute font size relative to the parent element's font size. Use of percentage values, or values in em's, leads to more robust and cascadable style sheets.
