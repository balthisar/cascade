Rename: This menu option allows you to safely rename the currently selected class. "Safely rename" means that the renaming dialogue box will consider the type of class the selector represents, and allow you to rename only the appropriate part.
 By holding down the Control key as you pop-up the contextual menu, the "Rename" option becomes "Expert Rename." You can then rename the entire selector, but the renamed class will then be processed by Cascade as a Custom/Expert class.
