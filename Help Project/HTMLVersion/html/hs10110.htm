Font-Family

Table of Contents Balthisar Software Site Attributes Panes


The font-family attribute specificies which font family is to be used to render the text. This property specifies a prioritized list of font family names and/or generic family names. To deal with the problem that a single font may not contain glyphs to display all the characters in a document, or that not all fonts are available on all systems, this property allows authors to specify a list of fonts, all of the same style and size, that are tried in sequence to see if they contain a glyph for a certain character. This list is called a font set.
To specify the font set in Cascade, you use the provided drop-down lists. The first two lists allow you to either select a font that exists on your system, or to type in the name of any font you wish to designate. The third list allows you to select a generic-family name, if you wish to do so.
