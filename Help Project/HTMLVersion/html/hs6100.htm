Cascade File Types

Table of Contents Balthisar Software Site Practical Use


 The following files are employed by the Cascade application. In some cases, your shell windows may be configured to hide extensions.

Extension File Type Description
 .WWS2 Cascade Document Balthisar Cascade document
.WWTD Cascade Stationary Templates for use with Cascade
 .WWST Cascade Swapfile Tempory file only. Will be erased.
.CSS Cascading Style Sheet(1) Text file containing CSS code.
.HTML HTML File(2) Text file containing HTML Markup.

Footnotes:
(1) Cascade does not change the file type description when it is installed, so it may appear differently on your system.
