Spin Editor Contextual Menu

Table of Contents Balthisar Software Site Cascade Overview


For spin editors that can show fractional numbers, the contextual menu to the left has been assigned. The purpose is to allow you to toggle the "whole-number" state of the spin editor in question. That way, values that should logically be whole numbers can be entered in a more sensical manner. For example, values that are set in pixels are whole numbers only.
