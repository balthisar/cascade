Text-Align

Table of Contents Balthisar Software Site Attributes Panes


The text-align property describes how inline content of a block is aligned.
Using the pop-up menu, you can select to have text justified to the left, right, center, or justify (full justification).
Using the edit-box, you can specify a string on which cells in a table column will align. This value applies only to table cells. If set on other elements, it will be treated as left or right, depending on whether direction is ltr, or rtl, respectively.
