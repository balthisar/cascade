 The class list consists of a list of classes that your CSS2 stylesheet contains. The class list is the central interface element to the classes in your stylesheet. It shows all of the HTML elements, classes, subclasses, pseudo-elements and -classes, custom "expert" classes, and @media descriptors. All of the classes in the class list can be rearranged into any order you may wish them to be; doing so affects the cascading order of the CSS2 file. You can rename classes in the class list; you can add new classes to the class list; and you can delete classes from the class list. Right-clicking on the class list brings up a contextual menu with options that apply to the selected class. In addition, several of the menu items in the main menu, as well as some of the shortcuts apply to individual classes selected in the class list.
If a class in the class list is tagged, then it is drawn with bold text. The export and preview functions allow you to apply their features to either all classes, or to tagged classes only. This gives you some flexibility during the design stages of your stylesheet.
You can rearrange the order classes are displayed in the class list. The display order is the CSS2 cascading order. To rearrange the order, simply drag-and-drop the selector you would move.
To copy a class in the current document to another document (or within the same document using this method), drag the selector you would copy, and drop it on the documents tabs' document you wish to copy to.
