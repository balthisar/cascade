@Media Descriptors:
  • Any Cascade selector or class that is created by the user specifically for implementing @media rules for multiple media stylesheets. The top-down cascade order of the class list affects which classes are present in the associated media rules. For example, if the class list has a meta-class called "@media aural," then any classes below it (up to the next @media rule) in the list will be exported nested within the @media aural { } rule.
