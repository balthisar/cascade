 Thank you for taking the time to try out Balthisar Cascade, the next incarnation of the premier internet development product from Balthisar Software.
Balthisar Cascade is intended for the creation of specification 2 Cascading Style Sheets (CSS2) for inclusion into HTML documents on networks such as the internet.
Balthisar Cascade includes most of the features an ameteur or professional HTML developer needs to create publishable hypertext documents. It follows the CSS2 specification as set forth by the World Wide Web Consortium. Cascade currently supports CSS2 strictly, i.e., it does not support proprietary extensions to CSS as used in Microsoft's Internet Explorer, and it does not support JavaScript style sheets at all.
Balthisar Cascade was previously known as WebWarrior Cascade. The WebWarrior moniker was dropped to avoid confusion with some existing products on the internet which already bear the "WebWarrior" name. The Balthisar moniker was added to avoid confusion with certain other products also named Cascade. So, the entire, proper name of this application is Balthisar Cascade, but within this document and the application itself, the name Cascade will suffice.

Some of Balthisar Cascade's features include:
• Creation of Classes
• Creation of Subclasses
• Creation of Pseudo-Classes and Pseudo-Elements
• Creation of ID's
• Creation of Element Classes
• Creation of Custom or Expert Classes
• The ability to set cascading order
• NEW! Managment of various media types.
• NEW! Allows multiple specification of CSS selectors.
• The ability to export data as an HTML file or a CSS2 file
• Many nice gems that make editing a Cascade document a pleasure.


 Some of Balthisar Cascade's limitations include:
• No ability to import existing CSS2 files exists. Due the various methods which the CSS2 specification allows rules to be defined, as well as time-to-market issues, Balthisar Cascade will not interpret such documents at this time. Tools will be developed which will allow this functionality.
• The ability to open WebWarrior Cascade documents is not present, due to the complete redevelopment of the file structure. Note: since this is now Balthisar Cascade, then Balthisar Cascade documents will open fine!
