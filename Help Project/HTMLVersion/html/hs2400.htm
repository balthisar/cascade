 Borland Delphi developers may be interested in including support for the *.WWS2 documents created by Balthisar Cascade. A free (compiled) Cascade Document class will soon be available which allows a Delphi 4.0+ application to generate CSS code from a Cascade document. As soon as the documentation is written, it will be available in beta-form.

Additionally, the editor itself will be made available as a complete VCL component for application designers who wish to support the editing of Cascade documents. Release of this component is pending the writing of the documentation. A license for unlimited use of the component will apply.

For those truly interested, this entire application consists of exactly:
  • One TCascadeDLOG component
  • One custom personalization component
  • One Data Module to house the components

     Please continue to monitor Balthisar Software for more details.
