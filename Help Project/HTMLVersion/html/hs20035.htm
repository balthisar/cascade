Contextual Class:
  • In the Cascade program, the term "class" is used to refer to any item in the class list. A contextual selector only works in the correct context. For example, TR P {...} only renders the specified attributes for <P> when it is nested within a <TR> element. This type of contextual selector implemented by Cascade is called a descendant selector. Other types of contextual selectors can be created using custom/expert classes.
