Text-Decoration

Table of Contents Balthisar Software Site Attributes Panes


The text-decoration property describes "decorations" that are added to an element. Values are as follows:

none
Produces no text decoration.
underline
Each line of text is underlined.
overline
Each line of text has a line above it.
line-through
Each line of text has a line through the middle
blink
Text blinks (alternates between visible and invisible).
