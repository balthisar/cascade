Custom Color Picker

Table of Contents Balthisar Software Site


 The custom color picker allows you to create a customized palette of your favorite or often-used colors. These colors, once on the palette, should be there again and again every time you use Cascade. Note: if you spend any time setting up your custom colors, ensure that you exit the color picker by choosing Okay, and not Cancel.
As the illustration shows, any time you hover the mouse over one of the custom colors, its name, if it has one, will be shown in a hint window. If the color has no matching name, the RGB value will be displayed instead.

To place the color of your choice into the custom colors grid, use either the RGB picker or the name picker to put the color of your choice into the composite color cell. Then, merely drag the composite color onto the custom color grid. You can drop the color on an empty cell, or drop it on an existing custom color to replace that color.
If you manage to drag the composite color over any of the other two color pickers, the other color picker will temporarily disappear to show the custom color picker, so that you can drop the color there. Once you are done with the drag-and-drop, the other color picker will reappear.
