@Media Descriptors

Table of Contents Balthisar Software Site Cascade Classes


 @Media descriptors are "meta" classes implemented by Cascade to allow the use of the @media rule as specified in the CSS2 specification.
@Media descriptors appear in the class list just like any other Cascade class. Like other classes, the descriptor can be dragged around and repositioned to affect top-down cascading order. Unlike other classes, @media descriptors cannot be edited in the attributes panes.
