Creating a New Document

Table of Contents Balthisar Software Site Practical Use


 A new document is automatically set up and ready to go for you as soon as you start Cascade, unless you started the application by opening a document. There are also three other ways you can start a new document.
  • From the main menu, select the File/New... command. Cascade will create a new document. If the file "default.WWTD"exists in the same folder as the application, then the new document will be based on that stationary.
  • Using the shortcut bar, select the icon for new document. If the file "default.WWTD"exists in the same folder as the application, then the new document will be based on that stationary.
  • Press the Ctrl-N key combination. Cascade will create a new, blank document.

     Once you've created a new document, you will probably need to create classes to modify. Browse through the Cascade Overview section for details.
