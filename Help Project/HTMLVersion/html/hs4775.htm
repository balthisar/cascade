Warning: During this process, Cascade will also remove any element which contains the value "text/css" anywhere in the tag, e.g., <STYLE TYPE="text/css">. This can lead to problems if you have that value in any HTML tags. In the just cited example, the <STYLE> tag would be removed, but not the corresponding </STYLE> terminator, nor the content of the tagset. If you know this will affect your HTML files, then don't use this feature. Instead, modify your HTML files manually.
