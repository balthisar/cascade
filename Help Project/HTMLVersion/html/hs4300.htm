 The preview pane allows you to take a look at the CSS2 text your stylesheet is generating. Normally, the preview pane will show only the class with which you are currently working, but you can choose to view all classes or all tagged classes (as shown to the left) by using the pop-up menu beneath the preview pane, or using the preview pane's contextual menu. If you choose the alternate views, the preview pane will revert back to displaying only the text of the currently selected class. If you want to copy text from the preview pane (for example, to paste it into an HTML document), you can do so by right-clicking in the preview pane, and using the contextual menu.
