Lineage
 Since the release of version 2.0b4, this help file is "complete," meaning we think it is done. Of course, nothing is every really complete, so we depend on your feedback to make changes and otherwise improve the help file.
This help file is not meant to be a replacement for adequate knowledge of the CSS2 specification. To make truly impressive stylesheets, some advanced knowledge of CSS2 is in order. To gain this knowledge, probably the best reference to CSS2 is from the people who wrote it: the World Wide Web Consortium. Much of the text related to CSS2 descriptions in this help file is from the official specification, and you can download the Adobe PDF version (suitable for printing) right here: http://www.w3.org/TR/1998/REC-CSS2-19980512/css2.pdf.

Hyperlinks
 The hyperlinks in this help have been color-coded to ensure that you do not connect to the internet unless you truly intend to. As is standard for Windows help files, internal hyperlinks are green and underlined. For example, Table of Contents will take you to the table of contents within this file.
There are also several links throughout which connect to internet URL's or other internet services. When these are clicked upon, the appropriate application will launch, and your operating system will attempt to connect to the internet, if it is not already connected. These "external" hyperlinks are blue, as in the fashion of HTML documents. For example, http://www.balthisar.com will take you to the Balthisar Software homepage on the internet.

Gender
 This help file uses the standard English convention of referring to unknown genders in the masculine. This is an attempt to simplify both the writing and clarity of this document. Its use is not intended to alienate nor offend members of any sex.
